# PRM-Tenbot

## Key Highlights

### 🔍 Knowledge Extraction & Structuring
Applied Natural Language Processing (NLP) techniques to extract and organize insights from unstructured data like our PRM Book, in PDF format by the [Project Management Institute](https://www.linkedin.com/company/projectmanagementinstitute/), creating a structured conceptual graph stored in Neo4j for efficient management and retrieval.

### 📊 Visualization & Insights
Enabled dynamic graph visualizations for intuitive exploration of project dependencies, offering users actionable insights and personalized recommendations.

### 💡 Data Retrieval with RAG
Leveraged Retrieval-Augmented Generation (RAG) to provide precise, context-aware answers, helping users navigate project risks and strategies with ease.

### 💬 Interactive Chatbot with Visual Support
Developed a conversational chatbot capable of providing not only accurate answers but also relevant figures, charts, and visual aids to enhance user comprehension of complex concepts and dependencies.

### 🌐 Web Development & Deployment
Built a user-friendly platform using React for the frontend and Flask for the backend, ensuring seamless interaction and efficiency for real-time exploration.
